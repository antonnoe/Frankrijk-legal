# 🇫🇷 Frankrijk-Legal

**Frankrijk-Legal** is een centrale hub ontworpen voor de Nederlandse gemeenschap in Frankrijk. Het project biedt juridische transparantie, compliance-informatie en diepgaande gidsen over complexe Franse regelgeving, zoals de bouwverzekeringen.

## 🚀 Project Status: Alpha (2025)
Gehost op GitHub & Vercel.

---

## 🛠 Tech Stack
- **Frontend:** HTML5, Tailwind CSS
- **Typografie:** Poppins (Koppen), Mulish (Broodtekst)
- **Design:** Custom Maroon (#800000) kleurenpalet met zachtgele accenten.
- **Hosting:** Vercel (Auto-deploy via GitHub)

---

## 📁 Bestandsstructuur

### 1. `index.html` (De Wegwijzer)
Een uitgebreide zoekmachine voor compliance van 100+ Franse bedrijven (o.a. Doctolib, SNCF, Orange). 
- **Functionaliteit:** Real-time filteren op naam of categorie.
- **Compliance:** Directe links naar Privacy Policies en Algemene Voorwaarden.
- **Community:** Snelkoppelingen naar Infofrankrijk.com en Nederlanders.fr.

### 2. `decennale.html` (De Dieptegids)
Een interactieve gids over de *Assurance Décennale* (10-jarige bouwverzekering) gebaseerd op de Franse Code Civil (Art. 1792).
- **Interactieve Checklist:** Controleer attesten van aannemers (voortgang wordt opgeslagen via `localStorage`).
- **Garantie Calculator:** Bereken direct de vervaldatum van de bouwgarantie.
- **Wettekst Integratie:** Directe Deep Links naar Légifrance.
